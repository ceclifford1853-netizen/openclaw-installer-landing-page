Privacy Policy

OpenClaw One-Click Installer

Last Updated: March 6, 2026

1. Introduction

OpenClaw Installer Project ("we," "us," "our," or "Company") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you visit our website at openclawwindowsinstaller.com and use our OpenClaw One-Click Installer software and related services.

We provide security-hardened Windows installer software designed to automate the setup of OpenClaw AI on Windows systems. Our installers incorporate the ClawJacked Shield security framework, which enforces local-only network binding and Windows Firewall rules to protect users from unauthorized external access.

Please read this Privacy Policy carefully. If you do not agree with our policies and practices, please do not use our services.

2. Information We Collect

2.1 Installation Information

When you run the OpenClaw One-Click Installer, we may collect:

  • Your Windows system architecture (32-bit or 64-bit)
  • Windows version and build number
  • Installation timestamp
  • Installation success or failure status

2.2 Telemetry Data (Optional)

If you enable telemetry, we may collect:

  • OpenClaw daemon uptime and status
  • Error logs and crash reports
  • Feature usage statistics
  • Performance metrics

Telemetry is disabled by default and requires explicit opt-in.

2.3 GitHub Integration

If you download the installer from GitHub, GitHub may collect analytics data as per their privacy policy.
