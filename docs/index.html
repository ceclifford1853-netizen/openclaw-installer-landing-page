Frequently Asked Questions

Is OpenClaw free?

Yes, OpenClaw is completely free and open source. There are no hidden costs or premium versions.

Do I need WSL2 to use OpenClaw?

WSL2 support is optional. OpenClaw works on standard Windows installations, but WSL2 integration provides enhanced functionality for developers using Linux tools.

What Node.js versions are supported?

OpenClaw supports Node.js 16.0 and higher. We recommend using the latest LTS version for best compatibility.

Can I use the portable version on a USB drive?

Yes! The portable version is designed specifically for USB drives and can run from any location without installation.

How do I verify the download?

Use the SHA-256 hash provided on this page. Compare it with your downloaded file using Windows PowerShell: Get-FileHash filename.exe

Is there support available?

Yes! Visit our GitHub repository for documentation, issue tracking, and community support.
